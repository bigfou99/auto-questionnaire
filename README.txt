# README for using the JavaScript function

To use the JavaScript function provided, follow the steps below:

1. Open the questionnaire in your web browser.

2. Open the console in your web browser.

   - For Chrome and Firefox, press `Ctrl + Shift + J` on Windows/Linux or `Cmd + Option + J` on Mac.

   - For Safari, go to `Preferences` > `Advanced` and check `Show Develop menu in menu bar`. Then, press `Option + Cmd + C`.

3. Copy and paste the JavaScript code from the 'auto_questionnaire.js'** file into the console.

   - To open 'auto_questionnaire.js', right click file and select edit.

4. Fill the questionnaire

   - Press `Enter` to fill all the questionnaire.

   - Press `Space to fill questions seperately` to fill all the questionnaire.

5. Fill out the questionnaire as usual.

   - Note that pressing the `Enter` key will automatically focus on the next question in the questionnaire.

   - The function is designed to work with radio buttons and checkboxes.

   - The function prevents the default behavior of the `Enter` key, so pressing `Enter` will not submit the form.

   - If the last question is reached, the `Enter` key will focus on the first question in the questionnaire.

   - You can use the left and right arrow keys to change the selection of the radio button.

6. Submit the questionnaire when finished.



# Chrome Extension Installation Guide

This guide will help you install the Chrome extension from the unpacked files.

## Prerequisites

1. Make sure you have downloaded the Chrome extension files.
2. Unzip the downloaded file if it is in a compressed format (e.g., .zip or .tar.gz).

## Installation Instructions

Follow these steps to install the Chrome extension from the unpacked files:

1. Open the Google Chrome browser.
2. Click on the three-dot menu icon in the top-right corner of the window.
3. Navigate to "More tools" > "Extensions."
4. In the Extensions page, enable "Developer mode" by toggling the switch in the top-right corner.
5. Click the "Load unpacked" button that appears when Developer mode is enabled.
6. In the file picker, navigate to the location where you extracted the Chrome extension files.
7. Select the folder containing the unpacked extension files and click "Open" or "Select Folder."
8. The extension should now be installed and appear in your list of extensions.

## Troubleshooting

If you encounter any issues during the installation process, make sure to:

1. Double-check the folder you selected in step 7 contains the correct unpacked extension files, including the `manifest.json` file.
2. Ensure that you are using a compatible version of the Google Chrome browser.
3. Check if there are any error messages in the Extensions page, and refer to the extension's documentation or support resources for assistance.

## Updating the Extension

To update the extension with new unpacked files, follow these steps:

1. Replace the existing unpacked extension files with the updated files.
2. Open the Extensions page in Google Chrome (follow steps 1-3 in the Installation Instructions).
3. Find the extension in the list, and click the circular arrow icon to reload the extension with the updated files.

Now you should have the latest version of the extension installed.

## Removing the Extension

To remove the extension from Google Chrome, follow these steps:

1. Open the Extensions page in Google Chrome (follow steps 1-3 in the Installation Instructions).
2. Find the extension in the list, and click the "Remove" button.
3. Confirm the removal in the dialog that appears.

The extension should now be removed from your list of extensions.
